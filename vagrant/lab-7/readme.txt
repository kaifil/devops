####### Webapp Folder ######

- Créez un dossier lab-7
- Déplacez-vous dans ce dossier
- Créez un vagrantfile afin de configurer 1 VM à partir des informations suivantes:
          - Image de base : <username>/nginx (vagrant box créé au lab-2)
	      - CPU : 1
	      - RAM : 1 Go
          - Ip fixe : 10.0.0.10
- Récupérer en local dans le dossier lab-7 l'application static-website-example
- Montez le code de l'application par le moyen de votre choix dans le dossier /usr/share/nginx/html de la VM
- Vérifiez que le site récupéré depuis github est bien accessible
- Arrêtez la VM puis supprimez la
