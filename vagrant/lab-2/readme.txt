######Lab-2   Créer un vbox ######
- Créer un dossier lab-2
- Déplacez-vous dans ce dossier
- Initialisez vagrant en utilisant la version 20190425.0.0 ubuntu/trusty64
- Démarrez la VM
- Connectez-vous en ssh
- Installez nginx
- Démarrez le service et activez son démarrage au lancement de la VM
- Créez une vbox à partir de cette VM
- Créez un compte sur vagrant cloud
- Publiez votre vbox sur vagrant cloud sous le nom <username>/nginx en version
  v1
