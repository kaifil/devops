####### Création d'un Vagrantfile ######

- Créez un dossier lab-3
- Déplacez-vous dans ce dossier
- Créez un vagrantfile afin de configurer la VM à partir des informations
  suivantes : - Image de base : Centos 7 By Geerlingguy
	      - CPU : 2
	      - RAM : 2 Go
- Variabilisez les paramètres indiquées ci-dessus
- Connectez-vous en ssh
- Installez nginx
- Arrêtez la VM puis supprimez la
